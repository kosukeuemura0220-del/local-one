Local ONE

Mac・Windows・iPhone・Android の間で、ファイル・動画・画像・GIF・.env・APIキーを、同じWi-Fi内で安全に送受信できるアプリです。

Apple Silicon (M1以降) / Intel 対応

🪟 Windows

Windows 10 / 11 (x64) 対応

インストーラー版をダウンロード ポータブル版をダウンロード

iPhone / Android からは、Mac または Windows でアプリを起動後に表示されるURLへブラウザでアクセスしてください（PWA対応）。

初回起動時の注意

このアプリは個人配布のため未署名です。OSの警告が出ますが、以下の手順で開けます。
  • Mac:「開発元を確認できないため開けません」と表示された場合は、アプリを右クリック→「開く」を選択してください。
  • Windows: SmartScreenの警告が出た場合は「詳細情報」→「実行」をクリックしてください。

使い方

  1. MacまたはWindowsでLocal ONEを起動する
  2. 表示されたQRコードを、もう一方の端末のカメラで読み取る（カメラがない場合は6桁コードを入力）
  3. 送信パネルからファイル・フォルダ・テキスト・.env・APIキーを選んで送信する
  4. 受信側で保存先や保存方法（クリップボードへコピー、フォルダへ保存など）を選ぶ

できること

  • 容量制限なしのファイル・フォルダ転送（GB級の動画にも対応、チャンク分割転送）
  • 転送後のSHA-256検証で破損チェック
  • .env / APIキー専用の「秘密モード」（履歴に残さない、TTL付きで自動失効）
  • 同一Wi-Fi内のデバイスを自動検出（mDNS）
Local ONE はローカルネットワーク専用のアプリです。インターネット経由での利用や、外部サーバーへのアップロードは行いません。
